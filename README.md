# Axsan - Small Business Accounting System

A complete, integrated small business accounting system built with React and Tailwind CSS.

## Features
- 📊 Real-time Dashboard with KPIs
- 💰 Cash Book Management
- 📥 Accounts Receivable with Aging
- 📦 Inventory Tracking
- 📤 Accounts Payable Management

## Live Demo
[View on Vercel](https://your-app.vercel.app)

## Local Development
1. Clone this repository
2. Open `index.html` in your browser
3. Start managing your business finances!

## Data Storage
All data is stored in your browser's localStorage for privacy and offline access.
